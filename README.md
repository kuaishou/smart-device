# 飞航终端设备
## 下载项目
```
git clone http://git.sinosafe.com.cn/dzswb/smart-device.git
```
## 安装依赖
```
yarn 或 npm i
```

### 运行项目
```
yarn dev 或 npm run dev
```

### 打包测试项目
```
yarn  build:test 或 npm run build:test
```
### 打包生产项目
```
yarn  build 或 npm run build
```
