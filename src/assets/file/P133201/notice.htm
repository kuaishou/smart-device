保险人告知事项

投保前，请投保人仔细阅读本《保险人告知事项》。保险责任、责任免除、退保、投保人及被保险人义务等内容详见产品条款。

1、保险人及承保范围：本保险由华安财产保险股份有限公司承保，只限为常住以下地区的人投保：北京、上海、广东、广西、福建、湖南、湖北、江苏、浙江、四川、重庆、辽宁、河南、天津、山东、安徽、陕西、云南、河北、内蒙古、江西、山西、黑龙江、贵州、吉林、海南、甘肃。本保险保障区域仅限中华人民共和国境内，不包括港澳台地区，外籍人士及港澳台人士不支持投保。

2、投保人：本保险投保人须为18周岁及以上具有完全民事行为能力人，为无民事行为能力人或限制民事行为能力人投保，投保人必须是被保险人的监护人。

3、被保险人：年龄为出生满30天（含）-65周岁（含）身体健康且符合约定的自然人。且被保险人职业类别不属于如下范畴：矿产资源作业、水上水下作业、高空作业、电力高压电、森林砍伐、化工、有毒及危险物质生产运输、水上运输、机械设备制造加工操作、金属/合金冶炼、海洋、特种养殖、救灾抢险、消防爆破、缉毒及防爆警察、拳击或足球职业运动员、潜水。

4、保险期间：本保险保险期为一年。

5、投保份数：本保险每一被保险人限投保一份，多购无效。

6、受益人：本保险的受益人为被保险人本人。

7、等待期：本保险的等待期为30天，扁桃腺、甲状腺、疝气、女性生殖系统疾病的检查与治疗等待期为120天。意外导致的保险事故或续保无等待期。

8、免赔额：（1）本保险交通工具意外伤害医疗责任免赔额100元；（2）本保险百万医疗险责任部分的免赔额为1万元，若被保险人在等待期后罹患本保险约定的重大疾病且在医院接受治疗的，则被保险人自该重大疾病确诊并接受住院治疗之日起发生的保险责任内的医疗费用不再扣除免赔额。

9、赔付比例：

交通工具意外伤害医疗特约：被保险人因发生主险合同责任范围的交通工具意外伤害事故，经治疗地社会保险经办机构指定医院治疗而支出的符合当地社会医疗保险主管部门规定的可报销的医疗费用，保险人在扣除100元免赔额后按80%的比例在保险金额内予以补偿，意外门、急诊赔付限额以300元为限。

一般医疗保险金和重大疾病医疗保险金的赔付比例：（1）若被保险人以参加社会基本医疗保险身份投保，并以社会基本医疗保险身份就诊并结算，赔付比例为100%；（2）若被保险人以参加社会基本医疗保险身份投保，但未以社会基本医疗保险身份就诊并结算，赔付比例为60%。若同时符合以下三个要件，赔付比例调整为80%：a.跨省或自治区或直辖市（仅限中国大陆境内，不包括境外及港、澳、台地区）异地就医；b.异地就医的医院无法使用被保险人参保所在地的社会基本医疗保险结算；c.被保险人已向参保所在地的医保经办机构申请报销，但参保地医保经办机构无法给予社会基本医疗保险报销。（3）若被保险人未以参加社会基本医疗保险身份投保，赔付比例为100%。

质子重离子医疗保险金的赔付比例：无论是否以社会基本医疗保险身份就诊，质子重离子医疗保险金赔付比例为100%，其中床位费最高给付上限为1500元/天。

附加恶性肿瘤院外特定药品费用医疗保险金的赔付比例：（1）若被保险人以参加基本医疗保险身份投保：a.如药品为基本医疗保险药品目录内且已经过基本医疗保险报销，赔付比例为100%；b.如药品为基本医疗保险药品目录内但未经过基本医疗保险报销，赔付比例为60%；c.如药品为基本医疗保险药品目录外，赔付比例为100%。（2）若被保险人未以参加基本医疗保险身份投保，则基本医疗保险药品目录内药品和基本医疗保险药品目录外药品赔付比例均为100%。

10、累计给付限额：一般医疗保险金、重大疾病医疗保险金、质子重离子医疗保险金、重大疾病异地转诊公共交通费用保险金累计给付金额以600万元为限，前述四项保险金的累计给付金额达到600万元时，保险人对被保险人的一般医疗保险责任、重大疾病医疗保险责任、质子重离子医疗保险责任、重大疾病异地转诊公共交通费用保险责任终止。

11、医院范围：本保险医疗相关保障限定经中华人民共和国卫生部门审核认定的二级及以上的综合性或专科公立医院，且仅限于上述医院的普通部。其中质子重离子医疗保险责任指定医疗机构仅限上海市质子重离子医院本部。

12、附加恶性肿瘤院外特定药品费用医疗保险责任：该责任的特定药品处方由保险人授权的第三方服务商上海镁信健康科技有限公司进行审核，药品购买流程适用条款中所列的“由保险人授权的第三方服务商审核的情形”。该责任的《药品清单》及具体购药流程详见电子保单中《恶性肿瘤院外特定药品服务手册》。

13、保险费缴付：本保险仅限选择一次性缴付全年保险费。

14、退保：在保险合同成立后，投保人可以书面形式通知保险人解除合同，但保险人已根据保险合同约定给付保险金的除外。投保人申请解除保险合同的，保险人仅退还未满期净保险费。

选择一次性缴付保险费，未满期净保险费=保险费×（1-m/n），其中，m为已生效天数，n为保险期间的天数，经过日期不足一日的按一日计算。

15、关于续保：本保险合同为非保证续保合同。除另有约定外，投保人可于保险期间届满前30日内申请续保，经保险人同意且投保人向保险人按照本保险合同的约定交纳保险费后，本保险合同于保险期间届满之次日零时起续保，保险人将重新签发保险单。续保本保险时，保险费适用续保保险费。 

保险人保留终止本保险合同续保的权利，当发生下列情形之一的，本保险合同不再接受续保：

（一）续保时被保险人的年龄超过100周岁；

（二）本保险产品统一停售；

（三）被保险人身故；

（四）保险人认为不符合续保条件的其他情形。

16、保险费率调整：本保险的保险费率会随着被保险人的年龄增长而改变。同时,保险人每年都会检视保险费率，使其反映保险人的整体理赔经验和医疗通胀等在内的一系列因素。保险人将根据本保险合同计算费率所依据的计算基础与实际情况的偏差程度，决定保险费率是否调整及调整幅度。本保险合同的费率调整适用于所有被保险人或同一投保年龄段、同一投保区域等某一类人群的所有被保险人。保险人进行保险调整后，投保人重新申请投保本产品时须按照调整后的保险费率支付保险费。若投保人不同意费率调整的，保险期间届满，保险人不再与投保人重新签订本保险合同。

17、适用条款：本保险适用条款为《华安财产保险股份有限公司交通工具意外伤害保险（2019版）条款》、《华安财产保险股份有限公司交通工具意外伤害保险（2019版）附加意外伤害医疗保险条款》、《华安财产保险股份有限公司住院费用医疗保险（2020A款）条款》、《华安财产保险股份有限公司附加恶性肿瘤院外特定药品费用医疗保险（A款）条款》。投保书为投保人、被保险人与保险人所订立保险合同的组成部分，在确认投保书之前，请详细阅读投保险种的保险条款并确认条款含义。

18、适用范围：本保险各项保险责任仅限于中华人民共和国境内（不含香港、澳门和台湾地区）。

19、告知义务：依据我国《保险法》的规定，投保人、被保险人应如实告知，否则保险人有权依法解除保险合同，并对于保险合同解除前发生的保险事故不负责任。

20、保险金申请：发生保险事故后，被保险人或受益人须及时（发生之日起五天内）凭电子保险单号向保险人报案，并提供相关证明和资料，保险人将按照条款约定履行给付保险金责任。

21、相关授权：如投保人投保此保险，视为投保人本人授权保险人出于提升服务质量之目的，合法地从任何单位、组织和个人获取有关的资料和证明，作为审核投保、处理理赔及提供与本保险有关其他服务的依据。

22、保险人已就投保人所投保保险项目的条款内容在相关互联网站页面的投保流程中设置投保人点击确认环节，由投保人确认以下内容：已阅读保险条款的全部内容，了解并接受包括免除保险人责任条款、投保与理赔规则、退保等在内的重要事项。投保人选择投保表示投保人已确认并认可相关内容。
